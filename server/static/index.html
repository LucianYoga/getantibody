Antibody A faster and simpler antigen written in Golang.

You can download, extract it somewhere and source the antibody.zsh file:

    You can also download an older version.

    You can script it somewhere: 

    $ wget -O antibody.tar.gz "http://antibody.elasticbeanstalk.com/latest/$(uname -s)/$(uname -m)"
$ mkdir /somewhere/antibody
$ tar xzvf antibody.tar.gz -C /somewhere/antibody
$ source /somewhere/antibody/antibody.zsh
$ antibody

    On OSX, you can install it using homebrew: 

    $ brew tap caarlos0/homebrew-antibody
$ brew install antibody
$ source "$(brew --prefix)/share/antibody.zsh"
$ antibody
